# LoFo

A beautiful, sleek, and extensive app that helps report lost and found items to our campus security and vice versa.

#### * NOTE: IIT Indore's insitute account is required for authentication.

To download, go to [releases](https://github.com/darrylad/LoFo/releases). For Android, tap the .apk file in Assets.

Or, for Android, [download v0.7.1-beta.47 here](https://github.com/darrylad/LoFo/releases/download/v0.7.1-beta.47/app-arm64-v8a-release.apk).

Created inside Google Developer Student Club, Indian Institute of Technology, Indore.
